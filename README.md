# Apointment App

1. Clonar el repositorio
2. Clonar el archivo .env.template y renombrar archivo .env
3. Ejecutar el comando `npm install` para instalar las dependencias
4. Verificar que el backend corra en el puerto 3001
5. Ejecutar el comando `npm run dev` para iniciar el servidor de desarrollo

## Stack

- Next.js
- Typescript
- Tailwind
- Shadcn
- Next-auth
- Zustand
